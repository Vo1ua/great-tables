Google Sheets Glossary

Plain-English definitions of the terms used across these tutorials — no jargon, just quick, clear explanations.

Cell Reference

A way of pointing to a specific cell's location, such as A1 or Sales!D5, so a formula can use its value instead of a fixed number.

See: Referencing a cell in another tab →

Named Range

A custom, readable name assigned to a specific cell or range of cells, such as CategoryList, used in place of a raw cell reference.

See: Using named ranges in dropdowns →

Data Validation

A Google Sheets feature that restricts what can be entered into a cell, commonly used to create drop-down lists.

See: Creating a drop-down list →

IMPORTRANGE

A function that pulls data from one Google Sheets file into a completely different Google Sheets file.

Formula Bar

The input field above the spreadsheet grid where the formula or value of the currently selected cell is displayed and edited.

Tab (Sheet)

One page within a Google Sheets file, shown along the bottom of the screen; a single file can contain many tabs.

Range

A group of two or more connected cells, written as a start and end cell separated by a colon, such as A2:A10.

Absolute Reference

A cell reference locked with dollar signs, like $A$1, so it does not change when the formula is copied to other cells.

Relative Reference

A standard cell reference, like A1, that automatically adjusts based on where a formula is copied to.

Pivot Table

A summary tool that groups and aggregates large sets of data into a compact, reorganizable table.

Conditional Formatting

A feature that automatically changes a cell's appearance, such as its color, based on the cell's value or a custom rule.

VLOOKUP

A function that searches for a value in the first column of a range and returns a matching value from another column in the same row.

#REF! Error

An error shown when a formula references a cell, row, column, or sheet that no longer exists.

See: Fixing the #REF! error →
