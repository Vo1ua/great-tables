How to Change the Default Font in Google Sheets

Google Sheets uses Arial as its default font, but you're not stuck with it. Since Sheets introduced customizable themes, changing the default font takes just a few clicks — and once set, it applies across the whole spreadsheet, not just one cell at a time.

Step-by-Step: Changing the Default Font

  1. Open your spreadsheet and click Format in the top menu.
  2. Select Theme.
  3. In the panel that opens on the right, click Customize.
  4. Find the Font dropdown and choose a new font from the list.
  5. Click Done in the top-right corner of the panel.
Every cell still using the theme's default formatting updates immediately to the new font — no need to select the whole sheet manually.

What Counts as "Default" vs. Manually Formatted Text

This is the part that trips people up: the Theme font only affects cells that haven't been given their own specific font already. If you previously selected a range and manually changed its font through Format → Font, that text keeps its own setting and won't change when you update the theme.

Changing More Than Just the Font

While you're in the Theme customization panel, you can also adjust:

  • Text color — the default color for regular cell text
  • Accent colors — used in charts and some UI elements
  • Header style — background and text styling for header rows, if you're using the built-in header formatting

Common Issues

Font Doesn't Change for Some Cells

Those cells almost certainly have a manually applied font from before the theme change. Select them, go to Format → Font, and choose "Reset to theme default" if available, or manually reapply the new font.

New Spreadsheets Still Use Arial

Theme settings are saved per file, not globally across your account. Each new spreadsheet starts with the default theme again unless you build it from a template that already has your preferred font set.

Frequently Asked Questions

Does changing the default font affect existing spreadsheets?

It updates any cells still using the theme's default font formatting in the current file, and applies to new sheets you add afterward. Text you've manually set to a specific font elsewhere is not affected.

Can I set a default font for every new spreadsheet I create?

Not directly. The Theme setting only applies per file. To reuse a font setup across new files, save a spreadsheet with your preferred theme as a template and create new files from it.

Why didn't the font change for some of my cells?

Cells that were manually formatted with a specific font using Format > Font keep that manual setting and won't follow theme changes. Only cells still using the theme's default font are affected.
