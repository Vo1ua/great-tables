How to Fetch Data from Google Sheets into a Google Form

It's a common mix-up: Google Forms is excellent at sending data out to a spreadsheet every time someone submits a response, but it has no built-in way to pull data in from a spreadsheet automatically. If you want a dropdown or multiple-choice question in a Form to be populated from a list you keep in Sheets, you'll need one of the two approaches below.

Method 1: Paste a Column as Answer Choices (Quick, One-Time)

Google Forms has a lesser-known feature: pasting a list of values directly into a choice field automatically splits it into separate options.

  1. In Google Sheets, select the column of values you want to use (for example, a list of department names).
  2. Copy it with Ctrl+C.
  3. In your Google Form, add or open a Dropdown or Multiple choice question.
  4. Click into the first empty "Option" field and paste with Ctrl+V.
Google Forms detects the multiple lines in your clipboard and automatically creates a separate option for each one, instead of pasting everything into a single field.

The limitation: this is a snapshot, not a live connection. If you add a new department to the sheet later, you'll need to repeat the copy-paste to update the Form.

Method 2: Apps Script (Keeps Choices Synced Automatically)

If the list in your sheet changes often, a short Apps Script can update the Form's choices every time the sheet changes, without any manual copy-pasting.

  1. Open your Google Sheet, then go to Extensions → Apps Script.
  2. Delete any placeholder code and paste in the script below.
  3. Replace the form ID, sheet name, and range to match your own file.
function updateFormChoices() {
  var formId = "YOUR_FORM_ID_HERE";
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Data");
  var values = sheet.getRange("A2:A20").getValues().flat().filter(String);

  var form = FormApp.openById(formId);
  var items = form.getItems(FormApp.ItemType.LIST);
  items[0].asListItem().setChoiceValues(values);
}
  1. Click Run once and grant the requested permissions.
  2. Optional: go to Triggers (clock icon) and add a trigger so this runs automatically, for example "On edit" or on a time-based schedule.

You can find the Form ID in its edit URL, in the long string of characters between /d/ and /edit.

Common Issues

Pasted List Shows as a Single Option Instead of Separate Ones

This usually happens if the copied cells weren't a single column, or if you pasted into the question title instead of an option field. Make sure you copy exactly one column and paste directly into an "Option" input.

Apps Script Fails With a Permissions Error

The first time you run any script that touches both a Form and a Sheet, Google requires you to authorize it manually. Click through the "Advanced" link in the warning dialog if the project isn't verified, since this is expected for personal scripts you wrote yourself.

Frequently Asked Questions

Can a Google Form automatically pull live data from a Google Sheet?

Not natively. Google Forms has no built-in feature that continuously syncs a question's choices with a spreadsheet range. A one-time paste updates the choices once; ongoing syncing requires Apps Script.

Do I need to know how to code to use the Apps Script method?

No prior coding experience is required. The script is short and only needs the sheet name and column changed to match your own file.

Will this also send Form responses back into the Sheet?

That is a separate, built-in feature. Under the Form's Responses tab, clicking the Sheets icon links responses to a spreadsheet automatically, independently of how the choices were populated.
