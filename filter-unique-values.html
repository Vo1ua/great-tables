Google Sheets: How to Filter Unique Values

When a column has repeated entries and you only need to see each value once — a list of customer names without duplicates, or the distinct product categories in an order log — the UNIQUE function does this in a single formula, without touching your original data.

The Basic Formula

Say column A contains a list of names, some repeated. In an empty cell, enter:

=UNIQUE(A2:A)

This returns every distinct value from that range exactly once, in the order it first appears — no manual sorting or deleting required.

Worked Example

A (Original)C (Formula Result)
AnaAna
DavidDavid
AnaBen
BenCarla
Carla

Even though "Ana" appears twice in column A, it shows up only once in the UNIQUE result.

Sorting the Results Alphabetically

UNIQUE returns values in the order they first appeared, not alphabetically. To sort the output, wrap it inside SORT:

=SORT(UNIQUE(A2:A))

Filtering Unique Values by Condition (FILTER + UNIQUE)

To get unique values that also meet a specific condition — for example, unique product names, but only for orders from a certain region — combine FILTER and UNIQUE in one formula:

=UNIQUE(FILTER(A2:A, B2:B="East"))

FILTER runs first, narrowing column A down to only the rows where column B equals "East." UNIQUE then removes any duplicates from that filtered result.

Unique Combinations Across Multiple Columns

UNIQUE isn't limited to a single column. Passing a multi-column range returns unique combinations of values across all of them together:

=UNIQUE(A2:B)

For example, "Ana, North" and "Ana, South" would both appear separately in the results, since the combination of name and region differs between them — even though "Ana" alone is repeated.

Common Issues

#REF! Error — "Array Result Was Not Expanded"

UNIQUE needs empty space below and to the right of the formula to spill its results into. This error means something (usually existing data) is blocking that space — clear the surrounding cells and try again.

Results Include an Unwanted Blank Value

If your source range includes empty cells, UNIQUE treats a blank as a legitimate value and lists it once in the results. Wrap the range in a check that excludes blanks, such as combining it with FILTER to skip empty rows first.

Frequently Asked Questions

Does UNIQUE change or remove the original data?

No. UNIQUE creates a new list in a different location without touching the source range, so the original data stays exactly as it was.

Can UNIQUE filter based on more than one column?

Yes. Passing a multi-column range to UNIQUE returns unique combinations across all of those columns together, not just unique values in a single column.

How do I sort the results of UNIQUE alphabetically?

Wrap the UNIQUE formula inside SORT, for example =SORT(UNIQUE(A2:A)), and the deduplicated list will be returned in alphabetical order.
