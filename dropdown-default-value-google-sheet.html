How to Set a Default Value for a Dropdown in Google Sheets

By default, a dropdown created with data validation starts out empty until someone makes a selection. That's not always what you want — if most rows should start as "Not Started" or "Pending," it's more efficient to pre-fill that value than to leave every new row blank. Google Sheets has no built-in setting for this, but there are two reliable ways to get the same result.

Method 1: Pre-Fill the Cell Directly (Simplest)

The most straightforward approach: after setting up your dropdown with data validation, just type the value you want as the default directly into the cell — as long as it exactly matches one of the dropdown's valid options, it will display normally as a selected chip.

  1. Set up your dropdown as usual through Data → Data validation.
  2. Click the cell (or select the whole column of cells).
  3. Type the exact text of your default option, for example Not Started.
  4. Press Enter.
This works because data validation only restricts what a user can type or select — it doesn't stop you from entering a matching value programmatically or manually ahead of time.

The limitation: this is a one-time pre-fill. If someone clears the cell, it goes back to blank, not automatically back to your default.

Method 2: A Self-Resetting Default (Formula-Based)

If you want the cell to automatically return to a default value whenever it's cleared — not just once, but every time — this requires a small formula trick using a circular reference.

  1. Go to File → Settings → Calculation.
  2. Set Iterative calculation to On, with max iterations set to 1.
  3. In your dropdown cell, enter a formula like: =IF(A1="", "Not Started", A1)

With iterative calculation enabled, this formula can safely reference its own cell: if the cell is empty, it shows "Not Started"; if a value has been entered (including a dropdown selection), it shows that value instead.

Circular Reference Warning

Without enabling iterative calculation first, this formula throws a circular dependency error, since the cell is referencing itself. This is expected — turning on iterative calculation with 1 iteration is what makes it work safely.

Which Method to Use

  • One-time setup, simple lists: Method 1 — pre-fill the cell directly
  • Cell should always fall back to a default when cleared: Method 2 — self-resetting formula

Frequently Asked Questions

Does Google Sheets have a built-in default value setting for dropdowns?

No. Data validation in Google Sheets has no native default value option, so dropdown cells start blank until manually pre-filled.

Will the pre-set default value get overwritten if the user picks a different option?

Yes, with the simple pre-set method. Typing the default value directly into the cell just gives it a starting point, and selecting a different dropdown option replaces it normally.

Why does the self-resetting formula show a circular reference error?

This happens because the formula references its own cell. Enabling iterative calculation in Google Sheets settings, with the iteration count set to 1, resolves it.
