How to Download One Tab from Google Sheets

Google Sheets makes it easy to download an entire workbook, but that's not always what you want — sometimes you just need one specific tab out of a dozen. The right method depends on the file format you need, since Google Sheets treats this differently for CSV, PDF, and Excel exports.

Quickest Option: Download as CSV or TSV

If a plain data file is all you need, this is a true one-click process:

  1. Click the tab you want to download, so it's the active sheet.
  2. Go to File → Download.
  3. Choose Comma-separated values (.csv).
Google Sheets automatically downloads only the currently active tab as a .csv file — there's no extra dialog or setting to configure. The .tsv (tab-separated values) option works exactly the same way.

Downloading as PDF

  1. Click the tab you want to export.
  2. Go to File → Download → PDF.
  3. In the export settings, set Export to "Current sheet" (rather than "Workbook").
  4. Adjust page size and orientation if needed, then click Export.

The Tricky One: Downloading a Single Tab as Excel (.xlsx)

Here's the part that trips people up: selecting Microsoft Excel (.xlsx) from the Download menu always exports the entire workbook, every tab included — there's no setting to limit it to just one sheet.

To get a single tab into a proper .xlsx file, use this workaround instead:

  1. Right-click the tab you want to export, at the bottom of the screen.
  2. Select Copy to → New spreadsheet.
  3. Google Sheets creates a new file containing only that one tab, and shows a confirmation with a link to open it.
  4. Open the new spreadsheet, then go to File → Download → Microsoft Excel (.xlsx).

Since the new file only ever had one sheet in it, the resulting .xlsx download contains exactly that tab — with formulas and formatting intact. You can delete the temporary spreadsheet afterward if you don't need it anymore.

Common Issues

My .xlsx Download Still Has All the Tabs

This is expected behavior for the direct Download menu — Excel exports always include the whole workbook. Use the "Copy to new spreadsheet" method above if you need a single-tab Excel file.

PDF Export Includes Every Tab

Check the Export dropdown in the PDF settings — it likely defaulted to "Workbook" instead of "Current sheet." Switching that setting exports only the active tab.

Frequently Asked Questions

Can I download just one tab on the mobile app?

The mobile app has more limited export options than the browser version. If you run into trouble, opening the file's link in a mobile browser instead of the app often exposes the full Download menu.

Does downloading one tab affect the original spreadsheet?

No. Downloading, and even the "Copy to new spreadsheet" workaround, never modifies the original file — both create a separate copy of the data.

Will formulas still work after downloading a single tab?

Formulas that only reference cells within that same tab will still work. Formulas that reference other tabs will break, since those tabs no longer exist in the downloaded file.
