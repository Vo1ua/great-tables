How to Split Text to Columns in Google Sheets

Google Sheets makes it easy to separate text into multiple columns. Whether you have names, addresses, email lists, or CSV data, you can split values using the built-in Split text to columns tool or the SPLIT formula.

For more details about supported delimiters and how the feature works, see Google's official Split text to columns documentation.

Using the Split Text to Columns Tool

  1. Select the cells containing your text.
  2. Click Data → Split text to columns.
  3. Choose the delimiter from the menu that appears.

Google Sheets instantly places each value into its own column.

The original data is replaced after splitting, so leave empty columns on the right before using the tool.

Example

Suppose column A contains full names.

A
John Smith
Emma Brown
Michael Johnson

Choose the Space delimiter.

The result becomes:

First Name Last Name
John Smith
Emma Brown
Michael Johnson

Split Text with the SPLIT Formula

If your data changes regularly, use a formula instead of the menu.

=SPLIT(A2,",")

This formula separates text wherever a comma appears.

You can also use other delimiters:

Formula Delimiter
=SPLIT(A2," ") Space
=SPLIT(A2,"-") Hyphen
=SPLIT(A2,"|") Pipe
=SPLIT(A2,";") Semicolon

Common Issues

Data Overwritten

If there is existing data in the destination columns, Google Sheets will overwrite it when using the Split text to columns tool.

Wrong Delimiter

If all text stays in one column, verify that you selected the correct delimiter. You may need to choose Custom and enter your own separator.

Formula Doesn't Split Correctly

Check that the delimiter inside the SPLIT function exactly matches the character used in your data.

Frequently Asked Questions

Can I split text without formulas?

Yes. Use Data → Split text to columns.

Can I split text by multiple characters?

Yes. The SPLIT function supports custom delimiters and multiple separator characters.

Does the SPLIT formula update automatically?

Yes. If the source cell changes, the formula recalculates automatically.
