How to Copy Comments in Google Sheets

If you've ever copied a cell in Google Sheets expecting the comment attached to it to come along too, you've probably noticed it just... doesn't. A regular Ctrl+C / Ctrl+V only copies the value or formula inside the cell — the comment stays behind. It's a common source of confusion, and there isn't a single obvious button to fix it. Below are two reliable ways to copy comments, plus a quick explanation of why this happens in the first place.

Why Comments Don't Copy Normally

A comment in Google Sheets isn't stored inside the cell the way a value or formula is. It's a separate object anchored to that cell, holding the comment text, the author, a timestamp, and any replies in the thread. Standard copy-paste only moves the cell's content — the comment layer is left out entirely. Once you know this, the two workarounds below make a lot more sense.

Method 1: Duplicate the Whole Sheet

This is the easiest option when you want an exact copy of an entire sheet, comments included — for example, when creating a backup or a reusable template.

  1. Go to File → Make a copy.
  2. In the dialog that appears, check the box labeled Copy comments.
  3. Click Make a copy.
Google Sheets creates a new file with every value, formula, format, and comment preserved exactly — including resolved comment threads.

The catch: this method duplicates the entire sheet or spreadsheet. It won't help if you only need to move comments from a handful of cells to a different location within the same sheet.

Method 2: Manually Recreate the Comment

If you only need to copy one or two comments — and don't need to preserve the original author or timestamp — copying the text manually is often the fastest option.

  1. Click the cell with the comment, then click the comment icon to open it.
  2. Select and copy the comment text with Ctrl+C.
  3. Click the destination cell.
  4. Insert a new comment (right-click → Comment, or press Ctrl+Alt+M).
  5. Paste the text with Ctrl+V and save the comment.

It's a bit more manual, but for one-off cases it takes less time than duplicating an entire sheet.

What About Copying Many Comments at Once?

If you regularly need to move comments between ranges — not whole sheets — a short Google Apps Script can automate the process, reading each comment from a source range and reapplying it to a destination range. This is worth exploring once you're comfortable with the two methods above and find yourself repeating the manual process often; we'll cover Apps Script basics in a future article.

Common Issues

Comment Missing After Make a Copy

This almost always means the Copy comments checkbox was left unticked in the "Make a copy" dialog. Redo the copy and make sure it's checked before clicking through.

Regular Copy-Paste Still Isn't Bringing Comments Along

This is expected behavior, not a bug — Google Sheets has no built-in shortcut for copying comments between individual cells. Use one of the two methods above instead.

Comments are easy to overlook until the moment you actually need to preserve one — and knowing these two methods means you won't lose important notes when reorganizing a spreadsheet or building a new template from an existing one.

Frequently Asked Questions

Does regular Ctrl+C / Ctrl+V copy comments in Google Sheets?

No. A normal copy-paste only copies the cell's value or formula — the comment is a separate object and stays behind.

Can I copy just one comment without duplicating the whole sheet?

Yes — open the comment, copy its text manually, then paste it into a new comment on the destination cell.

Does "Make a copy" preserve comment threads and replies?

Yes, as long as the "Copy comments" checkbox is ticked before clicking "Make a copy" — it preserves the full thread, including replies.
