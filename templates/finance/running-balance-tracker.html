Running Balance Tracker Template

Copy the structure below into a blank Google Sheet, paste in whichever formula fits your case, and it's ready to use.

1. Column Headers

Row 1 of your sheet — copy this row exactly:

Date | Income | Expense | Balance

Put each word in its own cell: A1 = Date, B1 = Income, C1 = Expense, D1 = Balance.

2. Sample Data (Rows 2–5)

Paste this in to see the template working immediately, then replace it with your own numbers:

DateIncomeExpenseBalance
7/120002000
7/3851915
7/105002415
7/152202195

D2 is typed directly (it's your starting balance) — every row after that uses a formula, not a typed number.

3. Choose Your Formula

Option A — Basic (drag down manually)

Type this into D3, then drag the fill handle down for each new row:

=D2+B3-C3

Option B — Fully Automatic (fills itself in)

Type this once into D2 instead, and it extends on its own as you add rows:

=ARRAYFORMULA(IF(B2:B="", "", SUM(B$2:B2)-SUM(C$2:C2)))

Option C — With a Fixed Starting Balance

If you're tracking an existing account, put your starting balance in cell G1 and use:

=$G$1+SUM(B$2:B2)-SUM(C$2:C2)
Not sure which one to pick? Option A is fastest to set up for a one-time tracker. Option B is better if you'll be adding rows often and don't want to drag formulas down every time.

Customizing It

  • Add a Category column (column E) if you want to break down spending later with a SUMIF by category.
  • Add a Notes column for context on unusual transactions.
  • Change the currency format: select column D → Format → Number → Currency.
  • Color-code negative balances: use conditional formatting to turn the Balance column red if it drops below zero.
