Can I Share Only One Tab of a Google Sheet?

Short answer: not natively. Google Sheets grants access at the level of the entire file — once someone is shared in, they can click through to every tab in the workbook, including ones with sensitive data, other clients' numbers, or internal notes. There's no built-in "share this tab only" button. That said, there are several practical workarounds, and which one is right for you depends on whether you need the data to stay live, stay editable, or just stay private.

Method 1: Hide the Other Tabs (Fastest, Weakest Privacy)

  1. Right-click each tab you don't want to share.
  2. Select Hide sheet.
  3. Click Share and set the recipient's role to Viewer.

Important Limitation

Hidden tabs are not actually removed from the file. Anyone with edit access can right-click the tab bar and choose Unhide sheet to see everything. This method only protects against view-only collaborators — never use it if you're granting edit access.

Method 2: Copy the Tab Into a New Spreadsheet (Most Reliable for a One-Time Share)

  1. Right-click the tab you want to share.
  2. Select Copy to → New spreadsheet.
  3. Open the new file and click Share as usual.

The new file contains only that tab — formulas, formatting, and data validation are preserved, and the rest of your original workbook stays completely private.

The trade-off: this is a snapshot, not a live connection. If the original data changes, you'll need to repeat the process or manually update the copy.

Method 3: IMPORTRANGE (Keeps the Shared Tab Live)

If the recipient needs to see updates automatically, pull the tab into a new file using a formula instead of copying it:

  1. Create a new blank spreadsheet.
  2. In cell A1, enter: =IMPORTRANGE("source_url", "TabName!A:Z")
  3. Replace source_url with your original file's URL, and TabName!A:Z with the tab and range you want.
  4. You'll see a #REF! error at first — click the cell and choose Allow access to connect the two files.
  5. Share this new file as normal.

This keeps the shared tab automatically in sync with the source, without exposing the original file. Just be aware it's one-way — edits made in the shared copy don't flow back to your original.

Method 4: Publish to the Web (View-Only, Public Link)

  1. Click the tab you want to share.
  2. Go to File → Share → Publish to web.
  3. Under "Link," select the specific sheet from the dropdown instead of "Entire Document."
  4. Click Publish, then copy the generated link.

Not for Sensitive Data

A published link is public — anyone who has it can view the tab, with no login required and no way to restrict access to specific people. Use this only for non-sensitive, view-only reporting.

Which Method Should You Use?

  • Quick internal share, trusted viewers only: Hide sheet
  • One-off report, no live updates needed: Copy to new spreadsheet
  • Ongoing client dashboard that needs to stay current: IMPORTRANGE
  • Public, read-only reporting: Publish to web

Frequently Asked Questions

Can Google Sheets share just one tab natively?

No. Google Sheets controls access at the level of the whole file, not individual tabs. Every method above is a workaround around this limitation.

Which method keeps data updating live?

IMPORTRANGE is the only method on this list that keeps the shared tab automatically updated, since it pulls live data from the source file into a separate spreadsheet.

Is hiding a tab a safe way to keep it private?

Only against viewers. Anyone with edit access can right-click the tab bar and choose Unhide sheet to see hidden tabs, so this method only works if collaborators have view-only access.
